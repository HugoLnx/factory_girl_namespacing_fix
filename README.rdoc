= factory_girl_namespacing_fix

Just a gem to fix the namespacing bug in factory_girl

== Installing

  gem install fatory_girl_namespacing_fix
  
== Using
	Just require fatory_girl_namespacing_fix after of require factory_girl
	Like that...
	
	require 'factory_girl'
	require 'fatory_girl_namespacing_fix'
	
	or, in rails....
	
	config.gem 'factory_girl'
	config.gem 'fatory_girl_namespacing_fix'
